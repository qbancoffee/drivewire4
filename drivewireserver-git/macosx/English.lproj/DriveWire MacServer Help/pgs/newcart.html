Creating a new cartridge

In DriveWire MacServer, cartridges are represented as files on the Mac OS X file system. A cartridge contains the extension .dsk or .os9, and are also known as virtual disks or disk images and can contain a Disk BASIC or NitrOS-9 file system that can be read by your CoCo.

To create a new cartridge:

  1. Choose Disk > Create Blank Cartridge.
  2. Type in a cartridge name in the file dialog window that appears.

    If you do not put a file extension on the name, it will have the extension of .dsk.

Once the cartridge is created, you can insert it into a virtual disk drive. The cartridge is empty, and therefore will need to be formatted either in Disk BASIC or in NitrOS-9.
