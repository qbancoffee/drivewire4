DriveWire MacServer Help

Creating a new document

In DriveWire MacServer, cartridges are represented as regular files on the Mac OS X file system, with the extension .dsk or .os9. Cartridges are also known as virtual disks or disk images and can contain a Disk BASIC, NitrOS-9 or other file system that can be read by your CoCo.

To create a new document:

  1. Choose File > New.
  2. A new DriveWire MacServer window will appear. It can be configured to connect to an additional CoCo system on a separate USB to serial convertor.
