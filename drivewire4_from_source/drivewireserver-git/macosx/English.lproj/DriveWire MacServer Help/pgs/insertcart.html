Inserting a cartridge

Inserting a cartridge into a virtual disk drive makes that cartridge available for your CoCo to use.

To insert a cartridge:

  1. Click on the blank cartridge area on the face of the virtual disk drive.
  2. A file dialog window appears. Select the desired cartridge located on the OS X file system.

    Only files with the extensions of .dsk.
