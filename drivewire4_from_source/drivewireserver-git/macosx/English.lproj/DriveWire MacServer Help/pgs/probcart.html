DriveWire MacServer Help

Problem inserting cartridges

If you are having difficulty inserting a cartridge into one of the virtual disk drives, consider the following:

  1. The virtual disk drive must be empty before a cartridge can be inserted. Eject any inserted cartidge that occupies the virtual disk drive bay before attempting to insert another cartridge.
  2. A cartridge file must have either .dsk or .os9 as the filename extension appended to it. The file browser will not allow you to select any other file.
